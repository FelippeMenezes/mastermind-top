# mastermind-top

A command-line implementation of the classic code-breaking game Mastermind, built as part of The Odin Project's Ruby Curriculum.

## Features
- **Dual Game Modes:** Play as the **Creator** (set a secret code for the computer to guess) or the **Guesser** (crack the computer's code).
- **Computer Intelligence:** Includes a logic-based guessing algorithm for the computer player.
- **Interactive CLI:** Clear feedback on every turn (Exact matches vs. Color matches).
- **Configurable Gameplay:** 12 turns to solve the 4-color secret code.

## 🛠 Tech Stack
- **Language:** Ruby
- **Concepts used:** Object-Oriented Programming (OOP), Classes & Modules, Logic Algorithms.
